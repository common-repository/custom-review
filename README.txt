== Description ==

This plugin gives functionality to gain more and more reviews on woocommerce stores products to the Store owners.

Review plays one of the most important roles in creating trust for the product and store on customer mind before buying the product itself. But there are very few number of the review on the product.

There is a very huge difference in both. Then how to gain more review. Here comes this plugin in action.

Custom Review allows store owners to add the review as many as they want to the product. Plugins create a menu to the admin dashboard to name a custom review in which there is an option to add review which leads to the page to add review.

These pages consist of several different inputs which help the owner to add reviews. Input Field is as follows.

* Select product - To select the product.

* Author name - Name of author which is displayed by product single page with review.

* Rating - Set of star to select the rating.(default 5)

* Select Date - Set the time of review.

* Enter text - Text for review.

* Upload Image - Image to display on comment as icon.



The review created through this, will appear on the default in the comment section. Comment section will have added a new column called Images which has all the images uploaded on the comment using these plugins as you can see in the screen shot.

If you wish to edit the review which is created through a custom review plugin, you can easily edit it by navigating through that review which is enlisted in the comments section.

After navigating to the review you can click on edit, this opens your review in the same format as an added. Make appropriate changes you can esily. After that hit Save.




== Installation ==


1. Upload `zwk_custom_review.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Click on custom review menu on admin dashboard and add review. Fill out the details and enjoy.
